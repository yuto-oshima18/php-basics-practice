# php-basics-practice

## 概要
COACHTECH HTML&CSS入門 「7-1-6: PHPの基礎 - ハンズオン演習」にて作成

## 使用技術
- Docker(nginx・PHP8.2)

## 学んだこと
- PHPにおけるechoの動作、文法、変数とデータ型、各種演算子の基本
- PHPファイルへのHTMLタグの利用方法

- 改行等のHTML部分の動作については目視以外にデベロッパーツールからHTMLの確認も実施。

- 追加演習

- 課題3：各変数の値を全ての条件を網羅できるよう都度変更し、動作確認。
